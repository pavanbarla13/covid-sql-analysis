# COVID-19 Global Trend Analysis (SQL Project)

## Objective
To analyze global COVID-19 trends using advanced SQL techniques including window functions, ranking, and growth percentage calculations.

## Dataset
Daily COVID-19 data including:
- Location
- Date
- New Cases
- Total Cases
- Total Deaths
- Population

Total Records: 43,000+

## SQL Concepts Used
- GROUP BY & Aggregations
- Window Functions (LAG)
- CTE (Common Table Expressions)
- Growth Percentage Calculation
- Ranking & Sorting
- Monthly Trend Analysis

## Key Insights
- Identified countries with highest infection rate
- Calculated daily growth percentage
- Detected global peak infection day
- Analyzed monthly global trend
- Ranked countries based on infection severity
